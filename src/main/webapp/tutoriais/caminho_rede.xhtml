Acessar caminho de rede Tutorial sobre como acessar um caminho de rede
  1. Para acessar um caminho de rede, primeiramente você deve anotar o caminho. (Ex.: \\servidor\pasta)
  2. Tomaremos como exemplo o caminho de rede do FED LOG:
  3. \\srv-arquivos.pamals.intraer\PAMALS\DT\CDCP\BIBLIOTECA-TECNICA\FED_LOG
  4. Para acessá-lo, digitaremos o caminho na tela de execução do windows acessado através do atalho do teclado + R
  5. Caso apresente alguma mensagem de erro, favor verificar bloqueios na rede com a informática local.
  6. Feito o acesso corretamente, a pasta com a publicação será aberta.
